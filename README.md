# פרויקט שרת אינטרנט ב-Python

ברוכים הבאים לרפוזיטורי של פרויקט שרת האינטרנט ב-Python. שרת זה פותח כחלק ממטלה אקדמית ומדגים עקרונות מרכזיים בתקשורת רשת, טיפול בבקשות HTTP, ואבטחת מידע בסיסית.

## סקירה כללית

שרת זה בנוי ב-Python ומיועד לטפל בבקשות HTTP נכנסות. הוא מסוגל להגיש קבצים סטטיים, לנהל חיבורים מרובים במקביל, ומיישם מנגנוני הגנה בסיסיים כמו חסימת Directory Traversal.

## תכונות מרכזיות

- **הגשת קבצים סטטיים:** השרת תומך בהגשת קבצי HTML, CSS, תמונות ועוד, מתוך תיקייה מוגדרת.
- **טיפול בחיבורים מקבילים (Concurrency):** תמיכה במספר לקוחות המחוברים בו-זמנית.
- **טיפול בשגיאות (Error Responses):** החזרת קודי סטטוס מתאימים בהתאם לבקשה:
    - `200 OK` לבקשות תקינות.
    - `400 Bad Request` לבקשות שאינן תקינות או חסרות.
    - `404 Not Found` לקבצים שאינם קיימים בשרת.
- **אבטחה:** מנגנון מובנה למניעת חציית ספריות (Directory Traversal), המונע גישה לקבצים מחוץ לתיקיית השורש של השרת.

## דרישות מוקדמות

כדי להריץ את השרת, יש לוודא שמותקנת סביבת העבודה הבאה:

- **Python:** גרסה 3.x מותקנת על המחשב.

## הוראות הרצה

כדי להריץ את השרת על המחשב המקומי שלך, פעל לפי השלבים הבאים:

1. פתח את חלון הפקודה (Terminal או Command Prompt).
2. נווט לתיקייה בה נמצאים קבצי הפרויקט.
3. הרץ את הקובץ הראשי של השרת (יש לוודא מהו שם הקובץ הראשי בקוד שלכם, למשל `server.py`):
    ```bash
    python server.py
    ```
4. השרת אמור כעת לרוץ ולהאזין לבקשות. ניתן לגשת אליו דרך הדפדפן בכתובת המקומית ( `http://localhost:8080`).

## קבצים מצורפים לפרויקט

- תיקיית `docs`: מכילה תיעוד שאלות קריאת קוד (`code-reading-Q.txt`) ומסמך מתמטי (`math.docx`).
- תיקיית `images`: מכילה תמונות לשילוב באתר.
- תיקיית `public`: תיקיית השורש ממנה השרת מחלק את הקבצים הסטטיים.
- קובץ `.gitignore`: להגדרת קבצים שאין להעלות ל-GitHub.
- תיקיית `security`: מכילה קובץ תמונה, לתוכן של תיקייה זו המשתמש לא מורשה לגשת.
